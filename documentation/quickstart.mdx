## Upload source files

Upload one or multiple files. You can upload a wide variety of file formats.  
Following file types are available to upload.

- Text files in TXT, CSV, DOCX, PDF, and MD formats.
- OCR-able JPEG, PNG, and PDF formats.
- Transcribable MP3, WAV format files.
- Transcribable YouTube videos.
- Crawlable webpages.
- Notion pages.
